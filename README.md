# 🧵 Textile Website

A modern, responsive textile product website built with **Next.js**. This project showcases textile products and services in an elegant and SEO-friendly manner, designed for both customers and business growth.

## 🚀 Features

- ✅ Responsive UI for desktop and mobile
- ✅ Product catalog with filtering
- ✅ Product detail pages
- ✅ Contact & inquiry form
- ✅ SEO optimized with meta tags and structured data
- ✅ Static Generation (SSG) & Server-Side Rendering (SSR)
- ✅ Optional admin dashboard for managing products

## 🛠️ Tech Stack

- **Frontend:** Next.js (React), Tailwind CSS / SCSS
- **Backend (optional):** Node.js / Express / Firebase / Strapi
- **Database (optional):** MongoDB / PostgreSQL / Firebase
- **Deployment:** Vercel / Netlify
